# justice-delayed-portal
# ⚖️ Justice Delayed Is Justice Denied

## About

This is a frontend web project based on the idea that justice should never be delayed. The website allows users to view legal cases, monitor their progress through different court levels, and simulate the process of escalating cases when delays occur.

The project was created to demonstrate how technology can be used to present legal information in a simple, interactive, and user-friendly way.

## Features

* View pending legal cases
* Search and filter cases
* Court escalation pipeline
* Add a new case
* Case details popup
* Live case statistics
* Responsive design

## How to Use

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Explore the features of the portal.

## Future Scope

* Connect with a real database
* User login and authentication
* Real-time case updates
* AI-based legal assistance
* Case report generation
